SmartVoter

A decentralized voting system simulation using blockchain technology to ensure transparency, immutability, and tamper-proof vote recording. Each vote is cryptographically secured and linked to previous votes, making unauthorized modification instantly detectable.

Security

SHA-256 cryptographic hashing ensures every vote block is immutable and verifiable.

Transparency

All votes are publicly visible in a blockchain chain eliminating hidden manipulation.

Integrity

Any tampering instantly breaks hash linkage and invalidates the entire chain.

Problem Statement

Traditional voting systems suffer from lack of transparency, tampering risks, duplicate voting, and centralized trust issues. There is a strong need for a secure and verifiable mechanism ensuring immutability and accountability.

Literature Review

Research highlights blockchain’s potential in transparent digital voting but most solutions require heavy infrastructure and smart contracts.

Core Functionalities

  • Unique Voter ID validation
  • Duplicate vote prevention
  • Real-time blockchain validation
  • Tamper simulation & detection
  • Live vote counting dashboard

System Methodology

Dataset / Input

Voter ID, Candidate Choice, Timestamp and Hash linkage.

Architecture

Block class, Blockchain class, SHA-256 hashing and validation logic.
